Modalidad de trabajo
- Asunto: Aclaración sobre el historial de commits y la autoría cooperativa del proyecto.
Con el objetivo de optimizar los tiempos y mantener un flujo de trabajo práctico y eficiente, optamos por centralizar las actualizaciones del repositorio en uno de nuestros integrantes. La producción del proyecto fue estrictamente cooperativa y equitativa en todas sus etapas: el diseño, la maquetación y el código se trabajaron en conjunto, rotando y consolidando los archivos a nivel interno a medida que avanzábamos. Una vez unificada y consensuada la versión final de cada jornada, se procedía a la subida en la plataforma.

Si bien el historial técnico de GitHub registra de forma lineal únicamente al usuario que ejecutó los commits, queremos ratificar que cada avance es el resultado del esfuerzo y la autoría de todos los miembros del grupo. Realizamos las últimas modificaciones en el repositorio con el fin de que la plataforma refleje correctamente esta coautoría en la sección de colaboradores.
